### **Assignment Summary**



##### Part A: City Library Management System

This part of the assignment implements a Python-based City Library Management System. Key features include:



Storing books and members using dictionaries.

Recording borrow/return events in a log.

Persistence: saving and loading the library state to/from a JSON file.

Implementing a configurable borrowing limit per member.

Overdue book detection.

A menu-driven command-line interface (CLI) for interactive use by a librarian.



##### Part B: Retail Transaction Insights

This part focuses on analyzing a retail transaction dataset using Python and the pandas library. The main functionalities include:



Robust CSV loading and date parsing.

Data cleaning and handling missing values.

Feature engineering (e.g., Year, Month, DayOfWeek, Season).

Extensive data visualization to understand distributions, trends, and patterns.

Analysis of customer behavior (average spend, payment preferences, items per store type).

Evaluation of promotion and discount impact.

Seasonality trends analysis.

An interactive CLI to perform analyses and generate plots.
